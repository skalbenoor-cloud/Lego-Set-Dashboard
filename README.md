# Lego-Set-Dashboard
🧱 Lego Set Explorer - Power BI Dashboard for Set Analysis 

A dynamic , interactive data visualization tool built to explore lego sets data from 1970 to 2022 - focusing on year wise trend, most popular themes, total set pieces and complexity, relationships between prices vs pieces and age distribution.

Description:
The Lego Set Explorer is an interactive Power BI dashboard that analyses lego sets across years, themes , pricing, pieces, age distribution. It highlights trends such as growth in lego production, different theme sets, relationship between price and complexity, enabling users to explore insights.

Tech Used:
The dashboard was built using following tools and technologies:
* 📊 Power Bi Desktop - Data visualization platform used for report creation.
* 📂 Power Query - Data transformation and cleaning for preparing the data.
* 🧠 DAX(Data Analysis Expressions) - Used for calculated measures, dynamic visuals and conditional logic.
* 📝 Data Modelling - Relationship estalished between tables. 

Data Source:
ℹ️ Lego Set Explorer - Brickset
LEGO sets released from 1970 to 2022, including details on each set's theme, pieces, recommended age, retail price, and image.

Features/Highlights:
1.Business Problem - The business lacks in identifying and analysing the solution for LEGO sets across themes, pricing, years,and complexity making it difficult to identify trends ,compare products and decision making.

Key Questions such as - How many LEGO sets have been released since 1970? Is there a noticable trend?Is there a relationship between the price of a set and its number of pieces? Which has been the most popular theme in each decade? are difficult to answer with raw data.

2.Goal of the Dashboard - To deliver an interactive visual tool that enables users to have insight into the lego sets and being able to understand and make it easy to view the yearly trends, different themes, validate pieces and prices, complexity and age distribution.

3.Walkthrough of Key Visuals:

  * Key KPI's(Top Metrics) - Total no. of sets : 4645, Average Pieces : 411, Average Price : 43$.
  * Slicers - Theme Group,Theme,Age Range an interactive slicers lets user filter all visuals by selected category.
  * Parameter - Max Price a filter control that defines the upper limit of the price range for LEGO sets displayed and allows user to view the price <= selected       max price in table.
  * Table Visual - Displays detailed information about each LEGO set including name, theme, age range, and price.
  * Observations - Provides insights about trends, popular themes, popularity, and target age groups.
  * Dashboard Design - Used lego themed background for better visual appeal and designed to be user friendly and interactive.
  * Dynamic Image - allows user to view the sets dynamically by selected field.
  * Buttons - Reset Filters clears all selected filters and Explore Sets helps user to navigate into selected data. 

4.Business Impacts and Insights:
   The business can use this dashboard to view the top performing themes, understand which age group has more sets, look into price distribution impact, it also makes easier for users to select the accurate set they are looking for based on theme ,age group and price.At the same time it also provides premium sets for advanced users. Overall this dashboard helps in identifying key trends which also contributes to increased sales and consistent revenue generation.
